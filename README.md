# PYTHON-BASED-HASH-CRACKER
A Python-based tool for cracking hashes, supporting MD5, SHA-1, SHA-256, and more. It works on all major operating systems and helps retrieve hashed passwords or data for security testing and auditing.
# Key Features
**Supports Multiple Algorithms:** Cracks hashes created with MD5, SHA-1, SHA-224, SHA-256, SHA-384, and SHA-512.  
**Works on Any OS:** Compatible with Windows, macOS, and Linux.  
**Dictionary Attack:** Uses wordlists of common passwords and phrases to crack hashes.  
**Brute Force Attack:** Attempts to break weaker hashes or short passwords through trial and error.  
**Customizable Settings:** Allows users to choose the hash type, attack method, and output format.  
**Autosave Feature:** Automatically stores cracked hashes and results for future reference.
